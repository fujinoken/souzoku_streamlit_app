# 相続関係説明図ジェネレーター Ver2.7

## Ver2.7 修正点

- 「相続人一覧・協議書」メニューを追加
- 相続になる人だけを抜き出した相続人一覧を表示
- 相続人一覧PDFを出力
- 遺産分割協議書PDFを出力
- 財産の分割内容を入力可能
  - 不動産
  - 預貯金
  - 有価証券・株式
  - 自動車
  - 動産
  - ゴルフ会員権
  - その他
- 債務の承継、後日発見財産、その他条項を入力可能
- 署名押印欄を相続人一覧から自動作成
- SQLite保存に財産情報・協議書条項を追加

## 注意

このアプリは書類作成補助です。
戸籍・住民票・登記簿謄本・印鑑証明書等と照合してください。
遺産分割協議書は事案により法的確認が必要です。
