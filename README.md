# 相続関係説明図ジェネレーター Ver2.2

## Ver2.2 修正点

- 被相続人の氏名・生年月日・死亡日・最後の本籍・最後の住所を出力
- 相続人の氏名・生年月日・死亡日・本籍・住所・続柄を出力
- 遺産分割状況（相続／分割／相続放棄／未定など）を入力・出力
- 作成日・作成者氏名・作成者住所を出力
- 被相続人と配偶者は二重線で接続
- 配偶者以外は一本線で接続
- PDF／PNG／画面プレビューに反映
- SQLite保存項目を拡張

## 起動方法

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Streamlit Cloud

Main file path は `app.py` です。

## 注意

このアプリは相続関係説明図の作成補助ツールです。
相続人確定、戸籍調査、法的判断の代替ではありません。
実務使用時は戸籍・住民票・除籍謄本等の確認資料と照合してください。
