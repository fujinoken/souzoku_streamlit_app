# 相続関係説明図ジェネレーター Ver2.5

## Ver2.5 修正点

- 孫・代襲相続人の入力欄を追加
- 子の代襲相続に対応
- 死亡した子の右側に孫・代襲相続人を表示
- 法定相続分の基本値に代襲相続を反映
- 死亡した子の相続分は「代襲者へ」と表示
- 孫には死亡した子の取得分を均等割で自動入力
- SQLite保存・読込・更新に孫データを追加

## 使い方

1. 第一順位：子に、死亡した子を入力します。
2. その子の「状態」を「死亡」にします。
3. 孫・代襲相続人の表で「親」に、死亡した子の続柄を選びます。
   例：長男、長女、二男
4. 法定相続分の基本値を入力すると、孫へ代襲相続分が入ります。

## 注意

この計算は基本パターンの補助です。
兄弟姉妹側の代襲、再代襲、半血兄弟姉妹、養子、欠格・廃除、特別受益、寄与分などは手動修正してください。

## 起動方法

```bash
pip install -r requirements.txt
streamlit run app.py
```
