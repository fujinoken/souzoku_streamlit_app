# 相続関係説明図ジェネレーター Ver3.3

## Ver3.3 修正点

- 「状態」「相続状況」に応じて、民法の基本相続分が自動で変動
- 死亡した人・相続放棄・対象外の人は相続分を自動で空欄化
- 死亡した子に孫がいる場合、子の相続分を孫へ代襲相続として自動配分
- 子系統がいる場合は父母・兄弟姉妹の相続分を自動で空欄化
- 子系統がいない場合は父母へ、父母もいない場合は兄弟姉妹へ自動移行
- 配偶者あり／なしで相続分を自動切替
- PDF／PNG／Excel／相続人一覧／遺産分割協議書機能は継続

## 自動計算の基本パターン

- 配偶者＋子系統：配偶者 1/2、子系統全体 1/2
- 子系統のみ：子系統で均等割
- 配偶者＋直系尊属：配偶者 2/3、直系尊属全体 1/3
- 直系尊属のみ：均等割
- 配偶者＋兄弟姉妹：配偶者 3/4、兄弟姉妹全体 1/4
- 兄弟姉妹のみ：均等割
- 配偶者のみ：1

## 注意

この計算は基本パターンの補助です。
半血兄弟姉妹、兄弟姉妹側の代襲、再代襲、養子制限、欠格・廃除、特別受益、寄与分などは手動調整してください。
