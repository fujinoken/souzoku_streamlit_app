# 相続関係説明図ジェネレーター Ver2.3

## Ver2.3 修正点

- 法定相続分の基本値を自動入力
- 入力済みの相続分を守る「空欄のみ入力」ボタン
- すべて基本値へ戻す「上書き」ボタン
- 入力後、各相続人の「相続分」欄で任意変更可能
- 基本パターン表をアプリ内に表示
- PDF／PNG／プレビューに相続分を反映

## 基本計算パターン

- 配偶者＋子：配偶者 1/2、子全体 1/2
- 配偶者＋直系尊属：配偶者 2/3、直系尊属全体 1/3
- 配偶者＋兄弟姉妹：配偶者 3/4、兄弟姉妹全体 1/4
- 配偶者のみ：1
- 子のみ：均等割
- 直系尊属のみ：均等割
- 兄弟姉妹のみ：均等割

## 注意

この計算は基本パターンの補助です。
代襲相続、半血兄弟姉妹、養子、欠格・廃除、特別受益、寄与分などは手動修正してください。

## 起動方法

```bash
pip install -r requirements.txt
streamlit run app.py
```
